## Convention

| 태그이름 | 설명                                       |
| -------- | ------------------------------------------ |
| feat     | 새로운 기능 추가                           |
| fix      | 버그 수정                                  |
| rename   | 파일이나 폴더 이름 및 클래스명,함수명 수정 |
| design   | 디자인 변경                                |
| test     | 테스트 코드, 리펙토링 테스트 코드 추가     |
| docs     | 문서 수정                                  |
